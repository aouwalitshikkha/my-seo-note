# Schema example 

| Schema Type        | Used On Which Pages    | Purpose                          | Should It Appear Often? |
| ------------------ | ---------------------- | -------------------------------- | ----------------------- |
| **Organization**   | All pages (site-wide)  | Defines brand entity & publisher | ✅ Yes (every page)      |
| **WebSite**        | All pages              | Defines the website entity       | ✅ Yes (every page)      |
| **WebPage**        | All pages              | Base page container              | ✅ Yes (every page)      |
| **AboutPage**      | About Us page          | Brand & trust information        | ❌ Only About page       |
| **ContactPage**    | Contact Us page        | Contact & support info           | ❌ Only Contact page     |
| **BlogPosting**    | Blog posts             | Editorial / opinion content      | ✅ On blog posts         |
| **Article**        | Reference articles     | Neutral informational content    | ❌ Limited use           |
| **CollectionPage** | Categories, archives   | Lists of content                 | ✅ Where applicable      |
| **BreadcrumbList** | All hierarchical pages | Navigation context               | ✅ Recommended           |
| **FAQPage**        | Pages with FAQs        | Q&A rich results                 | ❌ Only if real FAQs     |
| **Person**         | Author pages           | Author identity                  | ❌ Only where needed     |
| **Product**        | Product pages          | Commerce & offers                | ❌ Only if selling       |
| **Review**         | Review pages           | Ratings & opinions               | ❌ Only with stars       |
| **HowTo**          | Step-by-step guides    | Task instructions                | ❌ Only for steps        |
| **ContactPoint**   | About / Contact pages  | Support info                     | ❌ Limited use           |
